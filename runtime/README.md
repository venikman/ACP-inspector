# ACP.Runtime (F#)

ACP.Runtime is the runtime SDK for ACP: transports, codec, connections, contrib helpers, and observability tags.

## Build

dotnet build runtime/src/ACP.fsproj

## CI policy

This repo intentionally does not include Azure Pipelines YAML. CI is configured externally; do not add `azure-pipelines.yml` here.

## Examples

See `runtime/examples/` for end-to-end usage.
